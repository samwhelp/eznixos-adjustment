

# lightdm

## Explore

run 

``` sh
dpkg -S /etc/lightdm/lightdm.conf 
```

show

```
lightdm: /etc/lightdm/lightdm.conf
```


run 

``` sh
dpkg -S /etc/lightdm/lightdm-gtk-greeter.conf
```

show

```
lightdm-gtk-greeter: /etc/lightdm/lightdm-gtk-greeter.conf
```
